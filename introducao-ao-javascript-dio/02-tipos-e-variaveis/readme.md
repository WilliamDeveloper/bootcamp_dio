###### conversao

- `String(45.8)` - converte para string
- `Number.parseInt('45.5')` - converte para inteiro
- `Number.parseFloat('45.5')` - converte para float

- `45.84422.toFixed(2)` - formata para 2 casas decimais

- `"".toLowerCase()` - tudo para minusculo
